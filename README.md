
# CCU_API
## A public API for CCU firmware.

## Installation

This repository comes with precompiled binaries in the [CCU_CAPI/precompiled](CCU_CAPI/precompiled) directory, but you can compile the module yourself using [build.windows.sh](CCU_CAPI/build.windows.sh) or [build.linux.sh](CCU_CAPI/build.linux.sh).

## Examples

Examples can be found in [CCU_CAPI/examples](examples) directory.
